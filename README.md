# Personal Finance Intelligence System (PFIS)

A backend-driven personal finance analysis system built using Flask and SQLite.
The application tracks income and expenses, analyzes spending patterns, detects
overspending risks, and provides personalized savings recommendations using
explainable rule-based logic.

## Features
- Income and expense tracking
- Category-wise spending analysis
- Spending percentage computation
- Overspending alerts
- Personalized financial suggestions

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS

## Motivation
This project was built to understand how financial data can be analyzed and
translated into actionable insights using simple, transparent rules similar
to entry-level banking systems.

## How to Run
1. Create virtual environment  
2. Install dependencies: `pip install flask`  
3. Run: `python app.py`  
4. Open `http://127.0.0.1:5000/`
